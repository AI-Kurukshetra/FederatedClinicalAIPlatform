# Project Definition

## Tech Stack
- Frontend/Backend Framework: Next.js
- Database & Auth: Supabase
- Deployment Platform: Vercel

## Deployment Plan
- Develop application using Next.js.
- Use Supabase for PostgreSQL database, authentication, and storage.
- Connect Supabase keys via Vercel environment variables.
- Deploy the Next.js app to Vercel with automatic CI/CD from Git.
